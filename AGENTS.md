# BJJ74 Site Target Notes

This repository is the Hugo target site for bjj74.ru.

## Boundary

- Keep site source here: `content/`, `layouts/`, `assets/`, `static/`, `hugo.yaml`, and public project docs.
- Do not add private AI prompts, SEO research, video pipeline state, session logs, or Claude/Codex agent assets here.
- Workbench materials live in `~/projects/grappling-chelyabinsk-vault`.
- Apply generated content here only after it is reviewed in the workbench.

## Commands

- Build: `hugo`.
- Dev server: `hugo server -D`.

## Publishing

- This target repository can be synced to its configured Git remote, but generated workbench state must stay out of this repo.
